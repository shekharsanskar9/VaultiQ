# 🔐 VaultiQ  
**Secure Password Manager Web App**

---

## 📌 Overview  
VaultiQ is a lightweight and secure password manager web application built using HTML, CSS, and JavaScript. It enables users to store, manage, and retrieve credentials efficiently while ensuring privacy through client-side storage.

---

## 🚀 Features  

- 🔒 Secure password storage using browser localStorage  
- 👁️ Password masking and visibility toggle  
- 📋 One-click clipboard copy functionality  
- ⚡ Dynamic UI updates without page reload  
- 🧠 Lightweight and fully browser-based (no backend required)  

---

## 🏗️ Tech Stack  

- HTML  
- CSS  
- JavaScript  
- Browser localStorage
